# Fix: 404 Error pada /login dan /profil

## Root Cause
Sidebar (`components/app-sidebar.tsx`) link ke `/profil` dan `auth-check.tsx` redirect ke `/login`, tapi kedua route tersebut tidak ada halaman (`page.tsx`).

## Fix

### 1. Buat halaman Login (`app/login/page.tsx`)
- Form login dengan email & password
- Validasi dengan zod + react-hook-form (konsisten dengan `lupa-password/page.tsx`)
- Simpan user ke localStorage `digital-habit-user`
- Link ke `/lupa-password` dan register
- Redirect ke `/dashboard` setelah login berhasil

### 2. Buat halaman Profil (`app/profil/page.tsx`)
- Tampilkan data user dari localStorage
- Form edit profil (nama, email)
- Tombol logout (clear localStorage, redirect ke `/login`)
- Layout dengan sidebar (konsisten dengan halaman lain)

### 3. Fix AuthCheck (`components/auth-check.tsx`)
- Set `authorized` ke `false` secara default (bukan `true`)
- Render loading/null sampai auth check selesai

### 4. Fix Logout Button (`components/app-sidebar.tsx`)
- Tambah `onClick` handler untuk clear localStorage
- Redirect ke `/login`

## Files to create

- `app/profil/layout.tsx` (dengan sidebar)
