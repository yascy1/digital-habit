# Fix: Browser Autofill on All Steps

## Problem
Autofill leaks across steps — name field got email, now password field gets autofilled too. Need to reset each form when its step mounts.

## Fix
**File:** `app/lupa-password/page.tsx` (line 52-57)

Extend the existing `useEffect` to also reset `resetForm` when step becomes `"reset"`:

```tsx
useEffect(() => {
  if (step === "verify") verifyForm.reset()
  if (step === "reset") resetForm.reset()
  // eslint-disable-next-line react-hooks/exhaustive-deps
}, [step])
```

## Verification
- `pnpm dev` → `/lupa-password`
- All three steps should have blank fields when they first appear
