# Dashboard Page Implementation Plan

## Overview
Membuat halaman dashboard di `/dashboard` dengan statistik cards, grafik visualisasi, dan insight digital habits.

---

## Step 1: Install Dependencies

```bash
cmd /c "npx shadcn@latest add card badge separator chart --yes"
```

Ini akan install:
- **Card** - container untuk stat cards dan sections
- **Badge** - indicator untuk trend naik/turun
- **Separator** - pembagi antar sections
- **Chart** - wrapper recharts (otomatis install `recharts` sebagai dependency)

---

## Step 4: Component Breakdown (semua di satu file)

Komponen dibuat sebagai local components di dalam `page.tsx`:

1. **StatCard** - Reusable card untuk ringkasan statistik
   - Props: `icon`, `label`, `value`, `trend`, `trendUp`
   - Render: Card > icon + label + value + Badge (trend)

2. **ScreenTimeChart** - Line chart recharts
   - ResponsiveContainer + LineChart + Line + XAxis + YAxis + Tooltip + CartesianGrid
   - Garis dengan area fill untuk visualisasi tren

3. **CategoryChart** - Donut/Pie chart recharts
   - PieChart + Pie + Cell + Tooltip + Legend
   - Center text menampilkan total jam

4. **InsightSection** - Card berisi daftar insight
   - Setiap insight: icon + teks + badge prioritas
