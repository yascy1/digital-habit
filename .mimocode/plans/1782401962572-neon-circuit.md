# Fix: Semua URL 404

## Diagnosis
- `next.config.ts` kosong — **tidak ada basePath** yang di-set
- Semua `page.tsx` sudah ada dan punya `export default` yang benar
- `pnpm build` sukses tanpa error, semua route ter-render
- Root cause: **`pnpm dev` kamu tidak pernah berhasil jalan** — port 3000 selalu occupied oleh process lain (background server), jadi tidak ada server yang melayani request → browser tampilkan 404

## Plan

### Step 1: Clean slate
1. Hapus folder `.next` untuk bersihkan cache build yang mungkin corrupt
2. Pastikan tidak ada node process yang masih occupy port 3000

### Step 2: Jalankan dev server
1. `pnpm dev` di terminal
2. Verifikasi server berjalan di http://localhost:3000

### Step 3: Verifikasi semua route
Buka di browser atau test via HTTP:
- `/` → 200
- `/login` → 200
- `/dashboard` → 200
- `/profil` → 200
- `/riwayat` → 200
- `/input-aktivitas` → 200
- `/lupa-password` → 200

## Files involved
- `next.config.ts` — tidak perlu diubah (tidak ada basePath issue)
- `.next/` — hapus untuk clean build
