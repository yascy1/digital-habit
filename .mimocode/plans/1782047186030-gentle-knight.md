# Plan: Redesign Riwayat Aktivitas Page

## Overview
Redesign the `/riwayat` page into a modern SaaS-style activity history table with modal detail popup, matching the user's exact UI specification.

## Step 0: Branch Setup
- Switch to branch `fitur-riwayat` (already exists)
- URL stays at `/riwayat`

## Step 1: Rewrite `app/riwayat/page.tsx`

Replace the current card-based grouped layout with a table-based layout. The page remains a `"use client"` component.

### Layout Structure
```
┌─ Sidebar (existing AppSidebar) ─┐  ┌─ Main Content ──────────────────────┐
│  Digital Habit logo              │  │  "Riwayat Aktivitas" title           │
│  Dashboard                       │  │  Subtitle                            │
│  Input Aktivitas                 │  │  ┌─ Top Bar ──────────────────────┐  │

└──────────────────────────────────┘  └─────────────────────────────────────┘
```

### Key Components to Build

1. **Period tabs** (Harian / Mingguan / Bulanan) — pill-style toggle buttons (reuse pattern from dashboard)
2. **Date range picker** — two `<input type="date">` fields styled as a compact grouped input (start → end)
3. **Filter button** — simple button with `IconFilter` (no dropdown needed for now, just the button)
4. **Activity history table** inside a Card:
   - Columns: Tanggal, Total Screen Time, Aktivitas Terbanyak, Aksi
   - Rows derived from grouped activities data
   - "Aktivitas Terbanyak" shown as colored badges:
     - Media Sosial → blue (`bg-blue-100 text-blue-700`)
     - Belajar/Kerja → red (`bg-red-100 text-red-700`)
     - Hiburan → orange (`bg-orange-100 text-orange-700`)
     - Gaming → green (`bg-green-100 text-green-700`)
   - Aksi column: Eye icon button (`IconEye` from tabler)
5. **Pagination** at bottom of card:
   - "Menampilkan 1-6 dari 18 data"
   - Rows per page selector (6, 12, 24)
6. **Detail Modal** (triggered by eye icon):
   - Dark transparent backdrop overlay
   - White card centered (~500px), rounded, shadow
   - Title: "Detail {Tanggal}"
   - Body: category breakdown list (e.g., "Media Sosial ... 2 jam 30 menit")
   - Footer: "Tutup" (secondary, closes modal) + "Edit Data" (primary blue, navigates to `/input-aktivitas`)

### Data Logic
- Use existing `getActivities()` from `@/lib/activities`
- Group activities by date, compute total screen time per day
- For "Aktivitas Terbanyak", find the category with highest total minutes per day
- State: activities array, period filter (Harian/Mingguan/Bulanan), date range, pagination (page, rowsPerPage), selected day for modal
- Period tabs and date range are visual/UX elements — the table always shows per-day rows from localStorage data

### Icons to Use
- `IconEye` from `@tabler/icons-react` for view button
- `IconFilter` for filter button
- `IconCalendar` for date picker
- `IconChevronLeft` / `IconChevronRight` for pagination
