### 3d. PDF generation function `handleExportPDF()`

Called when user confirms the date range in the popover.

**Logic:**
1. Parse `exportDateFrom` and `exportDateTo` to `YYYY-MM-DD` strings.
2. Get all activities via `getActivities()`.
3. Filter: `activity.date >= from && activity.date <= to` (string comparison works for ISO format).
4. If empty → `toast.error("Tidak ada data pada rentang tanggal ini")`, return.
5. Sort by `date` descending.
6. Get profile via `getProfile()` for user name.
7. Create `new jsPDF("p", "mm", "a4")`.
8. Draw header section (title, subtitle, info line, 3 summary boxes).
9. Draw table via `autoTable`.
10. Add page numbers in `didDrawPage` callback.
11. `doc.save("riwayat_YYYY-MM-DD_YYYY-MM-DD.pdf")`.

**Header section details:**
- Title: "Digital Habit" (font size 18, bold)
- Subtitle: "Laporan riwayat aktivitas screen time" (size 12)
- Info line: `{fullName} | {dateFrom formatted} - {dateTo formatted} | Dicetak: {timestamp}`
- Three summary boxes drawn with `doc.roundedRect()` or `doc.rect()`:
  - "Total durasi": sum all `durationHours*60 + durationMinutes`, format as "Xj Ym"
  - "Total entri": count of filtered activities
  - "Kategori dominan": category with highest total duration, formatted as capitalized + dash→space

**Category label helper:**
```ts
function formatCategory(cat: string): string {
  return cat.replace(/-/g, " ").replace(/\b\w/g, c => c.toUpperCase())
}
```

**Duration formatting helper:**
```ts
function formatDuration(hours: number, minutes: number): string {
  return `${hours}j ${minutes}m`
}
```

**autoTable config:**
- `head`: [["Tanggal", "Kategori", "Durasi", "Catatan"]]
- `body`: sorted filtered activities mapped to rows
- Column widths:Tanggal ~30mm, Kategori ~35mm, Durasi ~25mm, Catatan ~90mm (fills remaining A4 width ~190mm minus margins)
- `styles`: fontSize 9, cellPadding 3
- `headStyles`: bold, fill color matching a neutral theme
- `columnStyles`: Catatan cell `{ cellWidth: 'auto', overflow: 'linebreak' }`

**Page numbers** in `didDrawPage`:
```ts
const pageCount = doc.getNumberOfPages()
for (let i = 1; i <= pageCount; i++) {
  doc.setPage(i)
  doc.text(`Halaman ${i} dari ${pageCount}`, doc.internal.pageSize.getWidth() / 2, doc.internal.pageSize.getHeight() - 10, { align: "center" })
}
```
