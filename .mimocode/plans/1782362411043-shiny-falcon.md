## Langkah 2: Buat `lib/stats.ts`

Shared utility untuk perhitungan stats. Dashboard dan PDF akan reuse fungsi ini.

```ts
import type { Activity } from "./types"

export const categoryLabels: Record<string, string> = {
  "media-sosial": "Media Sosial",
  "belajar-kerja": "Belajar / Kerja",
  hiburan: "Hiburan",
  gaming: "Gaming",
  lainnya: "Lainnya",
}

export interface CategoryStat {
  key: string
  label: string
  totalMinutes: number
  percentage: number
}

export interface PeriodStats {
  totalMinutes: number
  activityCount: number
  categories: CategoryStat[]  // sorted by percentage desc
}

// Hitung total screen time + breakdown kategori dari array aktivitas
export function computeStats(activities: Activity[]): PeriodStats

// Format menit → "X jam Y menit"
export function formatDuration(totalMinutes: number): string

// Hitung rentang tanggal untuk periode
export function getPeriodRange(
  period: "Harian" | "Mingguan" | "Bulanan"
): { from: string; to: string }

// Filter aktivitas berdasarkan periode
export function filterByPeriod(
  activities: Activity[],
  period: "Harian" | "Mingguan" | "Bulanan"
): Activity[]
```

**Logic `computeStats`:**
- Iterasi activities, jumlahkan `a.durationHours * 60 + a.durationMinutes` per kategori
- Hitung persentase: `(kategoriMenit / totalMenit) * 100`
- Sort categories by percentage descending
- Return `{ totalMinutes, activityCount, categories }`

**Logic `getPeriodRange`:**
- `Harian` → from = today, to = today
- `Mingguan` → from = Senin minggu ini, to = Minggu minggu ini
- `Bulanan` → from = tgl 1 bulan ini, to = akhir bulan ini
- Format: `YYYY-MM-DD`

**Logic `filterByPeriod`:**
- Panggil `getPeriodRange()` dapat `{ from, to }`
- Filter `activities` where `a.date >= from && a.date <= to`

## Langkah 3: Buat `lib/pdf-export.ts`

```ts
import jsPDF from "jspdf"
import autoTable from "jspdf-autotable"
import type { Activity } from "./types"
import type { PeriodStats } from "./stats"
import { categoryLabels, formatDuration } from "./stats"

export function generateRiwayatPDF(
  periodLabel: string,      // "Harian" | "Mingguan" | "Bulanan"
  periodRange: { from: string; to: string },
  stats: PeriodStats,
  activities: Activity[]
): void
```

**Struktur PDF (urutan atas → bawah):**

### 3a. Header
- Y=20: Judul `"Digital Habit — Laporan Riwayat"` (bold, size 16)
- Y=30: `"Periode: 18 Jun – 24 Jun 2026"` (size 11, abu-abu)
  - Format tanggal: gunakan `toLocaleDateString("id-ID", { day: "numeric", month: "short", year: "numeric" })`

### 3b. Ringkasan (Y=45)
- Baris 1: `"Total Screen Time: X jam Y menit"`
- Baris 2: `"Jumlah Aktivitas Tercatat: N"`

### 3c. Breakdown Kategori (Y=65)
- Heading: `"Breakdown Kategori"` (bold, size 13)
- Untuk setiap kategori di `stats.categories` (sudah sorted by % desc):
  - Label kategori di kiri (x=15)
  - Progress bar di tengah (x=80, width=90):
    - Track: `doc.setFillColor(229,229,229)` + `doc.rect(80, y, 90, 6, 'F')`
    - Fill: `doc.setFillCOlor(100,100,100)` + `doc.rect(80, y, 90 * (pct/100), 6, 'F')`
  - Persentase di kanan (x=175): `"XX%"`
  - Spacing antar baris: 12 unit

### 3d. Tabel Data Mentah (setelah breakdown)
- Gunakan `autoTable(doc, { ... })`
- Columns: `['Tanggal', 'Kategori', 'Durasi', 'Catatan']`
- Rows: mapping `activities` (sort by date desc) → `[formattedDate, categoryLabel, formatDuration, notes]`
- Konfigurasi: theme 'striped', headStyle fillColor primary

## Langkah 4: Edit `app/riwayat/page.tsx`

### 4a. Implementasi Filter Periode
- Import `filterByPeriod`, `getPeriodRange` dari `lib/stats.ts`
- Ganti `filteredActivities` useMemo:
  ```ts
  const isManualDate = dateFrom !== "" || dateTo !== ""
  
  const filteredActivities = useMemo(() => {
    if (isManualDate) {
      return activities.filter(a => {
        if (dateFrom && a.date < dateFrom) return false
        if (dateTo && a.date > dateTo) return false
        return true
      })
    }
    return filterByPeriod(activities, activePeriod as "Harian"|"Mingguan"|"Bulanan")
  }, [activities, activePeriod, dateFrom, dateTo, isManualDate])
  ```

```ts
function handleExportPDF() {
  const range = isManualDate
    ? { from: dateFrom, to: dateTo }
    : getPeriodRange(activePeriod as "Harian"|"Mingguan"|"Bulanan")
  const stats = computeStats(filteredActivities)
  generateRiwayatPDF(activePeriod, range, stats, filteredActivities)
}
```
